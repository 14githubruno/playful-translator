Löffelsprache

Löffelsprache, in order to exist, needs German language. That said, the required input has to be written in German, so that you can produce proper Löffelsprache.


The rules are the following: each vowel is doubled and in between 'lew' is added. Examples:

1) Löffelsprache -> Löffelsprache -> Lölewöffelewelspralewachelewe -> lölewöffelewelspralewachelewe
2) Das ist ein Wort -> dalewas ilewist eleweilewin wolewort


As above, you can enter a single word or multiple ones, otherwise one or more sentences or gibberish: as you wish.


The final translation is completely lowercase.
